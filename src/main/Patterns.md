## String Operations
Character frequency maps - Helps solve most string & character list related problems.

### Character frequency maps
- create map (character key, frequency number), iterate through characters list & add frequency to value of the character key.
- For ASCII only characters, use 128 bit array, use index as ASCII value of character & increase frequency at that index.

#### Problems 
- Anagram detection (rearrange charactres, eg. listen & silent)
- Finding duplicates
- character validation eg. email contains certain string
- Pattern matching
- Palindrome
